07

Design Principles

Glass Over Solid
Cards are never opaque. Every surface is translucent, referencing the navy depth beneath.
Metric-First
The largest, highest-contrast element on any card is always the primary data value. Labels follow.
Semantic Color
Each metric domain has a dedicated color. Never mix colors within a single metric category.
Mono for Meta
DM Mono is reserved for labels, timestamps, tags, and technical values. DM Sans carries all primary content.
Light Weight Hero
Hero metrics (timers, big totals) use font-weight 300 to feel effortless. Titles use 700 for contrast.
Restrained Motion
Animations are subtle — 0.2–0.5s ease, translate only. No bounce, no scale, no dramatic transitions.
